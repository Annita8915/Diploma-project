Элементы JavaScript

JavaScript – это язык программирования, который добавляет интерактивность на ваш веб-сайт (например: игры, отклик при нажатии кнопок или при вводе данных в формы, динамические стили, анимация).
Эта статья поможет вам начать работать с этим захватывающим языком и даст вам представление о том, на что он способен.
JavaScript невероятно универсален и дружелюбен к новичкам. Обладая большим опытом, вы сможете создавать игры, анимированную 2D и 3D графику, полномасштабные приложения с базами данных и многое другое!

Базовая информация по JavaScript:


Основной тег

<script>

Программы на JavaScript могут быть вставлены в любое место HTML-документа с помощью тега <script>

Ключевое слово

let

Пример: let message

Переменная – это «именованное хранилище» для данных. Мы можем использовать переменные для хранения товаров, посетителей и других данных.

Типы данных

number

Числовой тип данных

Числовой тип данных (number) представляет как целочисленные значения, так и числа с плавающей точкой. Существует множество операций для чисел, например, умножение *, деление /, сложение +, вычитание - и так далее. Кроме обычных чисел, существуют так называемые «специальные числовые значения», которые относятся к этому типу данных: Infinity, -Infinity и NaN.

string

Строковый тип данных

Строка (string) в JavaScript должна быть заключена в кавычки.
В JavaScript существует три типа кавычек.

  • Двойные кавычки: "Привет".
  • Одинарные кавычки: 'Привет'.
  • Обратные кавычки: `Привет`.

    • boolean

    Булевый тип данных

    Булевый тип (boolean) может принимать только два значения: true (истина) и false (ложь). Такой тип, как правило, используется для хранения значений да/нет: true значит «да, правильно», а false значит «нет, не правильно».

    null

    Специальный тип данных

    Специальное значение null не относится ни к одному из типов, описанных выше. Оно формирует отдельный тип, который содержит только значение null.

    undefined

    Специальный тип данных

    Специальное значение undefined также стоит особняком. Оно формирует тип из самого себя так же, как и null. Оно означает, что «значение не было присвоено». Если переменная объявлена, но ей не присвоено никакого значения, то её значением будет undefined.

    object

    Специальный тип данных

    Тип object (объект) – особенный. Все остальные типы называются «примитивными», потому что их значениями могут быть только простые значения (будь то строка, или число, или что-то ещё). В объектах же хранят коллекции данных или более сложные структуры.
